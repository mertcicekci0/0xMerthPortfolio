# x402 Portfolio (0xMerth)

High-end, minimalist personal portfolio website inspired by the "x402" design language. Built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Typography-Driven**: Playfair Display (Serif) & Space Mono.
- **Minimalist & Brutalist**: Stark black and white color palette.
- **Interactive**: Framer Motion animations and hover effects.
- **Web3 Accents**: Dot-matrix patterns and code visualizations.

## Tech Stack

- [Next.js](https://nextjs.org) (App Router)
- [Tailwind CSS](https://tailwindcss.com)
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide React](https://lucide.dev)

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
